# BwMe (Budget With Me)

A mobile budgeting application that helps users track expenses, visualize spending, and map where their money goes using OpenStreetMap.

---

## Description

BwMe is an Android application developed in Java using Android Studio. It is designed to simplify personal finance management by allowing users to log expenses and instantly visualize them on a map. Each transaction is tied to a location, giving users a real-world perspective of their spending habits.

The app also includes a dashboard for recent expenses, a statistics page with charts for financial insights, and a profile section where users can manage their budget and recurring expenses.

---

## Getting Started

### Dependencies

* Android Studio (latest version recommended)
* Java Development Kit (JDK 8 or higher)
* Android SDK
* OpenStreetMap API (for map integration)
* Internet connection (for map loading)

---

### Installing

1. Clone the repository:

   ```
   git clone https://github.com/Digbabby/bwme.git
   ```

2. Open the project in Android Studio

3. Allow Gradle to sync dependencies

4. Connect an Android device or start an emulator

---

### Executing program

1. Build the project in Android Studio
2. Click Run
3. Select your device/emulator

Once launched:

* Add an expense → it appears on the map
* View recent transactions on the home page
* Check analytics in the statistics page

---

## Help

**Common Issues:**

* Map not loading:

  * Check internet connection
  * Ensure OpenStreetMap services are accessible

* App crashes on startup:

  * Rebuild project (`Build > Rebuild Project`)
  * Check for missing permissions (location, internet)
